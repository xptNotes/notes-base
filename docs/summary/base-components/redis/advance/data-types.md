# Redis数据结构与对象



> 在redis数据库中每个键值对key-value都是有对象组成的,其中key都是字符串对象,value对象类型有五种:
>
> 字符串对象、列表对象、哈希对象、集合对象、有序集合对象.
>
> 这五种数据对象是提供给用户直接使用的,其底层实现主要依赖以下8种数据结构:
>
> 简单动态字符串、字典表、双向链表、跳表、整数集合、压缩列表、快速列表
