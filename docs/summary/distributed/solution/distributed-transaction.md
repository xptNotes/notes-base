# 分布式事务理论和实践



分布式事务解决方案分为两种模式

- 强一致性分布式事务解决方案
- 最终一致性分布式事务解决方案



其中强一致性分布式事务解决方案主要有三种模型:DTP模型、2PC模型、3PC模型.

最终一致性分布式事务解决方案也主要有三种解决方案:TCC解决方案、可靠性消息最终一致性解决方案、最大努力通知型解决方案.
